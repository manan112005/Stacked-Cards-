🎴 Interactive Card Stack 

A modern stacked card interface built with HTML and CSS, featuring a clean layered design and smooth visual styling. This project highlights how simple frontend techniques can create attractive and reusable UI components.

🚀 Features

- Layered / stacked card layout
- Clean and minimal design
- Smooth visual depth effect
- Lightweight (no frameworks)

🛠️ Tech Stack

- HTML5
- CSS3

▶️ How to Use

1. Download or clone the repository
2. Open the project folder
3. Run "index.html" in your browser
